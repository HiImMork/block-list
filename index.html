About

This is a website that allows you to manage (and create) filter lists for AdGuard, uBlock Origin, or any other adblocking applications that uses AdGuard's syntax. This tool allows you to load a list (in plain text format), add new rules in the list then manage it by sorting the list, removing duplicates and empty lines, as well as add and remove strings that will make the rules more powerful.

This website also include a built-in character counter to help you create custom regular expression (regex) easier although not automatically. But it helps! At least for me, that's why I added it.
